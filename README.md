# Usage

## 개발 환경 세팅

```shell
yarn
husky install # 허스키(pre-commit library) 설치
husky add .husky/pre-commit "lint-staged --no-stash --verbose && yarn typecheck" # 허스키 명령어 세팅

```

- 윈도우의 경우 "yarn lint-staged --no-stash --verbose && yarn typecheck"를 사용

## 테스트 서버 실행

```shell
yarn dev
```

## 빌드

```shell
yarn build
```
